Session 정보 일괄 등록

엑셀에서 내용을 복사-붙여넣기하여 입력하여 주시기 바랍니다.

  • 세션번호 : 각 세션을 구분하기 위한 고유 값으로 입력 (서브 세션 정보를 엑셀로 일괄 등록 시 매칭 정보로 사용할 값)
  • Room : 시스템에 등록된 룸 이름과 동일하게 입력 (띄어쓰기까지 동일하게 입력 필요)
  • Time : 시스템에 등록된 시간 정보와 동일하게 입력 (띄어쓰기까지 동일하게 입력 필요)
  • 좌장/패널/기타패컬티/지정토론 : 국문(입력용) 이름으로 입력 (패컬티 메뉴에 등록된 정보와 일치하도록 입력 / 패컬티 메뉴에 미등록 시 등록 불가)
  • 카테고리 1 : 시스템에 등록된 카테고리명과 동일하게 입력 (띄어쓰기까지 동일하게 입력 필요)
  • 카테고리 2 : 텍스트로 입력
  • KR/EN : KOR 또는 ENG 로 입력
